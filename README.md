# 🎓 Student Social Media & Mental Health Analytics Dashboard

## 📌 Project Overview

This Power BI project analyzes student social media usage, mental health, stress levels, lifestyle habits, academic behavior, and country-wise trends. The project transforms raw student data into meaningful insights through interactive dashboards and data-driven storytelling.

## 🎯 Dashboards Included

- Executive Overview Dashboard
- Social Media Analytics Dashboard
- Mental Health Analytics Dashboard
- Lifestyle Analytics Dashboard
- Country Analytics Dashboard
- Drill Through Analysis Page

## 🛠️ Tools & Technologies Used

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- KPI Cards
- Drill Down
- Drill Through
- Conditional Formatting
- Interactive Slicers
- Business Storytelling

## 📈 Key Insights

- Students spend an average of 5.08 hours daily on social media.
- Instagram is the most frequently used platform among students.
- A significant number of students fall into High and Very High Stress categories.
- The average Mental Health Score is 6.23.
- Sleep patterns and study habits influence student well-being.
- Student behavior and mental health vary across different countries.
